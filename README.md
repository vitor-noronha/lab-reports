# lab-reports

Repositório de laboratórios práticos em Cybersecurity, construídos para desenvolver e documentar competências técnicas com evidências reais de configuração, ataque e defesa.

---

## Laboratórios

### [soc-lab](soc-lab/)

Laboratório completo de SOC doméstico focado em duas competências principais:

**Resposta a Incidentes** — prática de todas as fases do ciclo (Detecção, Contenção, Erradicação e Recuperação) com cenários reais de ataque, playbooks estruturados e medição de SLA conforme padrões de um SOC.

**Hardening** — implementação de controles de segurança em servidores Linux, Windows Server, firewall pfSense e serviços AWS, baseado nos frameworks CIS Benchmark e DISA STIG.

| Tecnologia | Função |
|---|---|
| pfSense + Suricata | Firewall perimetral e IDS de rede |
| Wazuh All-in-One | SIEM, HIDS e correlação de eventos |
| Kali Linux | Simulação de ataques (Metasploit, Hydra) |
| Ubuntu Server 22.04 | Servidor alvo Linux |
| Windows Server 2022 | Servidor alvo Windows / Active Directory |
| Ubuntu Desktop | Estação do analista SOC |
